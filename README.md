# codelens-golang-navigate

[![CI](https://github.com/hu553in/codelens-golang-navigate/actions/workflows/ci.yml/badge.svg)](https://github.com/hu553in/codelens-golang-navigate/actions/workflows/ci.yml)

- [License](./LICENSE)

A Visual Studio Code extension which adds CodeLens actions and hover links for navigating through Golang code.

## Features

- CodeLens actions for navigating through Golang code
- Hover links for navigating through Golang code

## Prerequisites

- Golang installed (https://golang.org/doc/install)
- Golang extension installed (https://marketplace.visualstudio.com/items?itemName=golang.go)
- Golang language server installed (https://github.com/golang/tools/tree/master/gopls)

## Configuration

- `codelensGolangNavigate.enableCodeLensActions`: Enable/disable CodeLens actions (default: true)
- `codelensGolangNavigate.enableHoverLinks`: Enable/disable hover links (default: true)
