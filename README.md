# codelens-golang-navigate

[![CI](https://github.com/hu553in/codelens-golang-navigate/actions/workflows/ci.yml/badge.svg)](https://github.com/hu553in/codelens-golang-navigate/actions/workflows/ci.yml)

- [Open VSX](https://open-vsx.org/extension/hu553in/codelens-golang-navigate)
- [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=hu553in.codelens-golang-navigate)

A Visual Studio Code extension which adds CodeLens actions and hover links for navigating through Golang code.

## Features

- CodeLens actions for navigating through Golang code
- Hover links for navigating through Golang code

## Prerequisites

- [Golang installed](https://golang.org/doc/install)
- [Visual Studio Code extension installed](https://marketplace.visualstudio.com/items?itemName=golang.go)
- [Language server installed](https://github.com/golang/tools/tree/master/gopls)

## Configuration

| Option                                         | Description                     | Default |
| ---------------------------------------------- | ------------------------------- | ------- |
| `codelensGolangNavigate.enableCodeLensActions` | Enable/disable CodeLens actions | `true`  |
| `codelensGolangNavigate.enableHoverLinks`      | Enable/disable hover links      | `true`  |
